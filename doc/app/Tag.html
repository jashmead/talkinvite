    class Tag

    Tags – Polymorphic, point up to basically everything:

    • Talks

    • People

    • Locations

    • Posts

    • Attachments

    • Tweets – used to track HashTags

    Fields

    1. tagable_type, mandatory, one of the relevant table names

    2. tagable_id, mandatory, one of the relevant table ids

    3. tag_type – default is '', normal tag, could be used for ratings, mandatory, one of allowed types

    4. tag – string, may be a de facto integer if this is a rating, mandatory

    Possible Fields

    1. comment/description – text

    2. person_id, integer – id of person creating the tag
