    Got Talk?

    Start points:

    1. home – for person, shows their talks

    2. talks/new – start a talk

    3. talks/search – look for them, with lots of canned searches

    4. talks/nearby – and recent – TBD

    5. topics/trending – recently interesting topics – TBD

    6. sitemap

    Let's Talk about X

    See searches for talks – and, once they are up, for topics

    1. key phrases – type key phrase, search web

    2. recent event – type key phrase, search web

    3. subject I've already talked about – tags associated with my talks or with me

    4. like a previous talk – my talks

    5. part of an ongoing conversation – talks by talk2talk

    6. associated with an organization (i.e. meetup), school – talks by talk2talk

    7. associated witeh a specific location – location to talks

    8. associated with a specific set of friends (or just one or two) – people to talks

    9. cool url for image, document, video on web – cut & paste

    10. personal event – type key phrase

    response, search for talk, pick talk(s), add if not present

    Search for Talks

    1. by all the categories above, many canned searches

    2. mark a set interesting, or add talk, or give up

    3. pick a talk functionality!*

    Join Talk

    perhaps even several

    1. if not signed in, sign in

    2. if no account, create & sign in

    Add Talk

    1. if not signed in, sign in

    2. if no account, create & sign in

    3. start the talk: see talks/new

    4. what – summary

    5. when – calendar

    6. where – maps, locations

    7. why – detailed description

    8. who – any restrictions (i.e. private, group)

    Check out topics

    1. topics/command

    2. sub set of tags

    Send out posts

    first, select set of related, search for people by patterns or rules

    second, post using:

    • messaging (internal)

    • email

    • twitter

    • facebook

    • other social media

    Schedule talk

    2. set defines

    Check out locations
